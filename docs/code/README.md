# QuickForge 代码文档

> 项目版本: 1.2.2 | 许可证: MIT

本文档按原始目录结构分类整理了 QuickForge 项目的所有代码文件。每页对应一个顶层目录或主要模块，包含文件清单、功能描述和关键代码要点。

---

## 目录索引

| 目录 | 说明 |
|------|------|
| [`src/`](src/README.md) | React 前端源代码（组件、Hooks、工具库） |
| [`server/`](server/README.md) | Node.js 后端服务器（路由、工具执行、存储） |
| [`bin/`](bin/README.md) | CLI 启动脚本 |
| [`scripts/`](scripts/README.md) | 构建和打包脚本 |
| [`skills/`](skills/README.md) | Agent Skills 定义 |
| [`.github/`](github/README.md) | GitHub CI/CD 配置、Issue 模板 |
| [根目录配置文件](root-configs.md) | 项目根配置文件 |

## 架构概览

```
┌─────────────────────────────────────┐
│          React 前端 (src/)           │
│  ┌─────────┐ ┌──────┐ ┌──────────┐ │
│  │components│ │hooks │ │   lib    │ │
│  └────┬────┘ └──────┘ └────┬─────┘ │
│       │                    │        │
│       ▼                    ▼        │
│  UI 组件 + 对话框      工具库 + API  │
└────────────────┬────────────────────┘
                 │ HTTP / SSE
                 ▼
┌─────────────────────────────────────┐
│      Node.js 后端 (server/)         │
│  ┌──────┐ ┌──────┐ ┌────────────┐ │
│  │routes│ │tools │ │agent-manager│ │
│  └──────┘ └──────┘ └────────────┘ │
│  ┌──────┐ ┌──────┐ ┌────────────┐ │
│  │utils │ │skills│ │  storage   │ │
│  └──────┘ └──────┘ └────────────┘ │
└────────────────────────────────────┘
```

前端通过 HTTP API 和 SSE 事件流与后端通信。后端提供存储、Agent 管理、工具执行和技能加载。
